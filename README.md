Local alignment search using suffix tree

code is in suffixTree/stree.go

To Build

```
cd suffixTree
go build
```

can now output to a graphviz plot to visualize the suffix tree

1. Install GraphViz
2. `./suffixTree | dot -Tpng -o graph.png`
